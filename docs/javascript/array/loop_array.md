# 遍历数组

1. map
2. foreach
3. for
4. filter
5. reduce

分别的使用场景：

1. `map`
   - 在拿到一个很大的数组，但我只需要其中一小部分数据时，可以用 map 将其取出来；
   - 需要将数组按照某种规则映射为另一个数组；
2. `forEach` / `for of`
   - 只需要进行简单的数组遍历；
3. `for of`
   - 需要对迭代器进行遍历时；
4. `filter`
   - 需要过滤出所有符合条件的项（返回的是由所有符合条件项所组成的数组）；
5. `map` 结合 `filter`
   - 先映射成新的数组，再从中过滤出符合条件的项。

> 千万不要因为过分追求性能，而忽略了语义和可读性。
