# 输入和输出

## 输出 `print()`

```python
>>> print('The quick brown fox', 'jumps over', 'the lazy dog')
The quick brown fox jumps over the lazy dog
```

## 输入 `input()`

```python
>>> name = input()
Michael
```

Python提供了一个 `input()`，可以让用户输入字符串，并存放到一个变量里。

```python
name = input('please enter your name: ')
print('hello,', name)
```
