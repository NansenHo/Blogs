### 浏览器三大进化路线

1. **应用程序 Web 化**
   云计算 和 HTML5 的普及 => 越来越多的应用转向浏览器/服务器（B/S）架构，且视频、音频、游戏等核心场景也都在往 Web 使用场景切换。
2. **Web 应用移动化**
   PWA
3. **Web 操作系统化**
   - 利用 Web 来创建一个纯粹的操作系统，比如 ChromeOS。
   - 浏览器的底层结构往操作系统架构方向发展。

浏览器已经逐步演化成了操作系统之上的“操作系统”。
