# 1.1 编译原理

## JavaScript 是一门编译语言

JavaScript 不是一门 “动态” / “解释执行” 的语言，而是一门**编译语言**。

但与传统的编译语言有区别。

### 传统编译语言

传统编译语言在源代码被执行前，会经历 ”编译“ 三步骤：

1. 分词/词法分析（Tokenizing/Lexing）
   - 将字符串分解成对编程语言来说有意义的代码块，这些代码块叫做词法单元（token）。
   - 程序 `var a = 2;` 会被分解成以下词法单元：`var` `a` `=` `2` `;` 。
   - 空格是否算作词法单元，取决于空格在这门语言中是否具有意义。
   - 
2. 解析/语法分析（Parsing）
3. 代码生成
