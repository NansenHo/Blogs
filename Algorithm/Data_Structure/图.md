# 图 G(V,E)

如果在一颗树的基础上，出现了闭合节点

图是由顶点集合和顶点之间边的集合组成的。

G(V,E) 
- G 表示图
- V 表示 G 中顶点的集合
- E 表示 G 中边的集合

如：
```js
// 4 => 1
// 5 => 3
        +----> 1
        |     / \
        |    2   3 <-+
        |   / \      |
        +- 4   5 ----+
```

图在前端用得较少。

但 Vite / Webpack 里的任意 module 里都有的 ModuleGraph 用的是图。

ModuleGraph 是内部的文件依赖图。

我们可以通过这个图来很快找到某个 module 是否重复加载了。
