# Vim 命令

| key | placement |
| --- | --- |
| :h | 打开官方文档 |
| Esc | 从输入模式切换到普通模式 |
| / | 正向查找 |
| ? | 反向查找 |
| /searchContent + `<CR>` | 高亮匹配项 |

| key | placement |
| :--- | --- |
| $ | the end of the line. |
| bb | 定位到单词开头 |
| ce | 删除单词，保留单词后面的空格，切换到输入模式 |
| cw | 删掉单词，保留单词后面的空格，切换到输入模式 |
| cc | 修改一整行，即删除一整行后，切换到输入模式 |
| dw | 删除单词和单词后面的空格，保持普通模式 |
| dd | 删除一整行，保持普通模式 |
| vit | 选择文本 |
| >G | 当前行锁进 |
| A | 定位到当行最后，切换到输入模式 |
